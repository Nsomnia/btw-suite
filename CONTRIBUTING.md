# Contributing to btw-suite

First off, thanks for taking the time to contribute! You are a true Chad.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or bugfix.
3. Ensure your code follows the "Chad" aesthetic (use `lib/chad.sh`).
4. Submit a pull request.

## Code of Conduct

Be a Chad. Don't be a non-Chad. Respect the Arch way.
